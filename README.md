## 저장소가 여러개로 나뉜 이유
> local 환경에서 개발을 할 당시에는 문제점을 인지하지 못하다가, AWS배포 작업을 들어가게 됐을때 Django버전/Python버전/모듈,패키지 버전들의 호환에 문제가 생겨 테스트 하다가 결국 3번째 저장소까지 가게 되었다.
